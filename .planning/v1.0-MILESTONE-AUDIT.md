## Critical Gaps

### 1. HUBI-03 — Cross-Tool Links (UNSATISFIED)

**Requirement:** Slots -> "Finanzberatung starten", T2 -> "Zur Finanzberatung"
**Evidence:** No cross-tool links found in any Slots or T2 templates. Searched templates/slots/ and templates/t2/ — zero references to Finanzberatung.
**Impact:** Users on Slots/T2 pages have no discovery path to Finanzberatung tool.
**Fix:** Add call-to-action links in Slots booking confirmation and T2 dashboard.

### 2. ADMN-03 — DSGVO File Deletion Path Bug (PARTIAL)

**Requirement:** Original files deleted after retention period; extracted values preserved.
**Evidence:** `finanz_dsgvo_service.py::execute_deletion()` constructs file path with spurious `'persistent'` segment: `{PERSIST_BASE}/persistent/{UPLOAD_DIR}/`. Upload service stores files at `{PERSIST_BASE}/{UPLOAD_DIR}/{session_id}/`. Path mismatch means `os.remove()` never finds files. `files_deleted` always returns 0. Database cleanup (anonymize filename, preserve extracted data) works correctly.
**Impact:** DSGVO compliance broken — physical files are never actually deleted from disk.
**Fix:** Remove `'persistent'` segment from path construction in `execute_deletion()`.

### 3. HUBI-02 — Access Control Scope (PARTIAL)

**Requirement:** All openers and closers can access the tool.
**Evidence:** `hub.py` lines 620-622: `return username in ('Admin', 'luke.hoppe')`. Access is hardcoded to two users instead of role-based.
**Impact:** Other openers/closers cannot see or access the Finanzberatung tool tile.
**Fix:** Replace hardcoded list with role-based check (e.g., check T1/T2 access lists).

### 4. PIPE-09 — Vector Similarity Search Not Exposed (PARTIAL)

**Requirement:** Similar documents can be found across sessions via vector similarity search.
**Evidence:** `FinanzEmbeddingService.search_similar()` method exists. ChromaDB stores embeddings correctly. But no route, template, or service ever calls `search_similar()`. The feature is implemented at the service layer but unreachable from any user interface.
**Impact:** Users cannot discover related documents across sessions.
**Fix:** Add search endpoint in sessions routes or integrate into session detail page.
