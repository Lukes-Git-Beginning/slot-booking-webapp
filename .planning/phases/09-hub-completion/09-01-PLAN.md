**4. Add before_request hook in app/routes/finanzberatung/__init__.py:**

Place the decorator and function at module level, after the blueprint object creation (line 21) but before `init_app()`. This ensures it fires for all sub-blueprints:

```python
@finanzberatung_bp.before_request
def require_finanz_access():
    """Restrict finanzberatung to openers, closers, and admins.
    Upload routes are excluded -- customers access those via token without login.
    """
    from flask import request, session, redirect, url_for, flash

    # Feature flag check
    if not finanz_config.FINANZ_ENABLED:
        flash("Finanzberatung ist nicht aktiviert", "warning")
        return redirect(url_for('hub.dashboard'))

    # Skip upload routes (customer-facing, token-based, no login required)
    endpoint = request.endpoint or ''
    if 'upload' in endpoint:
        return None

    # Require login for all other routes
    user = session.get('user')
    if not user:
        return redirect(url_for('auth.login'))

    # Role check
    from app.routes.hub import has_tool_access
    if not has_tool_access(user, 'finanzberatung'):
        flash("Kein Zugriff auf Finanzberatung", "error")
        return redirect(url_for('hub.dashboard'))
```

IMPORTANT: Do NOT exclude SSE routes from the guard. SSE is for logged-in consultants only. Only exclude upload routes.

IMPORTANT: The `from flask import session` here refers to Flask session (not SQLAlchemy). This file already imports from flask at line 14, so just add session to that import line.
