  <name>Task 1: Create public upload route (CSRF-exempt, token-validated)</name>
  <files>app/routes/finanzberatung/upload.py</files>
  <action>
Create `app/routes/finanzberatung/upload.py` with sub-blueprint `upload_bp = Blueprint('finanz_upload', __name__)`.

This blueprint is CSRF-exempt (handled in Plan 01's __init__.py). NO `@require_login` on any route -- customers are not logged in.

Routes:

1. `GET /upload/<token_value>` -- Upload page
   - Call `upload_service.validate_token(token_value)`
   - If invalid/expired: render `finanzberatung/upload_expired.html` with error_message
   - If valid: get customer name from `token.session.customer_name`
   - Get remaining uploads: `token.max_uploads - token.upload_count`
   - Get token expiry as ISO string for client-side countdown
   - Render `finanzberatung/upload.html` with: customer_name, token_value, remaining_uploads, expires_at, max_file_size_mb (from FinanzConfig)

2. `POST /upload/<token_value>/submit` -- Handle file upload (AJAX endpoint)
   - Validate token again (re-check -- token may have expired since page load)
   - If invalid: return JSON `{error: true, message: '...'}` with 403
   - Get file from `request.files['file']`
   - If no file: return JSON error 400
   - Call `upload_service.validate_file(file)` for magic byte validation
   - If invalid: return JSON `{error: true, message: '...'}` with 400 (German message from service)
   - Call `upload_service.store_file(file, token.session_id, token.id, file.filename, detected_mime)`
   - Return JSON: `{success: true, document: {id, original_filename, file_size, mime_type, status, created_at}}`
   - NOTE: Plan 04 will add SSE event publishing here after successful upload. For now, just store the file.

3. `GET /upload/<token_value>/status` -- Token status check (AJAX, for client-side polling)
   - Returns JSON: `{valid: bool, remaining_uploads: int, expires_at: iso_string}`
   - Used by client to update UI without page reload

Error handling: All errors return JSON with German messages. Use `logging.getLogger(__name__)`.
Log each upload: `logger.info(f"Document uploaded: {original_filename} for session {session_id}")`.

Instantiate service at module level:
```python
upload_service = FinanzUploadService()
```
